# 🐸 Doodle Jump — pygame OOP

## Встановлення та запуск

```bash
pip install pygame
python generate_assets.py   # генерація звуків (лише один раз)
python main.py
```

## Керування
| Клавіша | Дія |
|---------|-----|
| ← / A   | Рух вліво |
| → / D   | Рух вправо |
| ENTER / SPACE | Старт / рестарт |
| ESC | Пауза / вихід з меню |
| M | Увімкнути / вимкнути звук |

## Типи платформ
| Колір | Тип | Опис |
|-------|-----|------|
| 🟩 Зелена | Звичайна | Стандартний стрибок |
| 🟦 Синя | Рухома | Рухається горизонтально |
| 🟧 Оранжева | Ламається | Розбивається та падає (звук!) |
| 🟨 Жовта | Пружинна | Супер-стрибок (звук!) |

## Архітектура (ООП)

```
Game
 ├── SoundManager      — завантаження WAV, відтворення
 ├── AssetManager      — кеш шрифтів
 ├── Camera            — вертикальне скролювання
 ├── Player            — герой, фізика, малювання
 ├── ParticleSystem
 │    └── Particle     — ефекти вибуху
 ├── PlatformFactory   — фабрика платформ
 └── Platform (base)
      ├── MovingPlatform
      ├── BreakablePlatform
      └── SpringPlatform
```

## Звуки (assets/)
| Файл | Подія |
|------|-------|
| `jump.wav` | Звичайний стрибок |
| `break.wav` | Ламання платформи |
| `spring.wav` | Пружинний стрибок |
| `fall.wav` | Смерть гравця |
